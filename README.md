### About Me

I'm a full stack software engineer who loves breaking down complex problems into bite-sized pieces until the whole thing clicks. Give me a whiteboard, a team, and a hard problem and I'm in my element. I build things across the stack - from cloud infrastructure and APIs to ML models and frontends.

- **Senior Engineer** building production health-tech systems with Python, FastAPI, Terraform, and AWS
- **BS Computer Science** from Brigham Young University (2025)
- **Creator of [NBA Oracle](https://nbaoracle.com)** - ML-powered NBA game predictions, live in production
- Division 1 Ultimate Frisbee alum at BYU and disc golf enthusiast

### What I'm Building

**At Work** - Designing and building the backend architecture for a health-tech platform:
- Architected a layered API (FastAPI) with service-provider-DAO patterns and dependency injection
- Production AWS infrastructure with Terraform (Aurora PostgreSQL, Lambda, Cognito, S3, CloudWatch)
- CI/CD pipelines with GitHub Actions, Docker, and self-hosted runners on EC2
- AI-powered health insights pipeline integrating OpenAI for NLP and data extraction
- Structured observability with Sentry trace correlation and custom logging libraries
